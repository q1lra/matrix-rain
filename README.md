# Matrix Rain
A grid of numbers that randomly updates and changes colors in the command line.

## Requirements
* Linux (Tested on Debian 13)
* Python 3
* Git

## Installation
```bash
git clone https://github.com/q1lra/matrix-rain.git
cd matrix-rain
python3 matrix-rain.py
```
